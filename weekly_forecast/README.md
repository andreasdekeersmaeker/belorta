# 🫑 Bell Pepper Weekly Forecast

Predicts total incoming weight (kg) across all vendors for the next 1–2 weeks.

## Quick start

```bash
pip install -r requirements.txt

# Train model (fetches real weather from Open-Meteo automatically)
python train.py

# Launch app
streamlit run app.py
```

## File layout

```
├── app.py                      ← Streamlit forecast app
├── train.py                    ← Model training script
├── requirements.txt
├── data/
│   ├── historical_weights.csv  ← year, week, total_kg  (your data)
│   └── weather_historical.csv  ← auto-generated by train.py
└── model/
    └── xgb_model.pkl           ← auto-generated by train.py
```

## Updating the model

Add new rows to `data/historical_weights.csv` then run `python train.py` again.

## Input data format

`data/historical_weights.csv`:
```
year,week,total_kg
2024,10,450000
2024,11,512000
```

The app also accepts CSV upload and manual entry in the sidebar.

## Weather

Auto-fetched from [Open-Meteo](https://open-meteo.com/) for **Mechelen, Belgium** (51.028°N, 4.480°E).
No API key required.
